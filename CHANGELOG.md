## [1.0.0] - 2024-03-21

### Added
- 初始版本发布
- 基础音频处理功能
  - 音频设备管理
  - 音频流采集和播放
  - 音频处理器接口
- 跨平台支持
  - macOS 支持
  - Windows 支持
  - Linux 支持
  - Android 支持
- 核心功能
  - WebSocket 客户端
  - 线程池管理
  - 配置管理器
- UI 组件
  - 主窗口
  - 设备设置界面
  - Agent 管理界面
  - 欢迎界面

### Dependencies

- SAMPLERATE

- GTest
